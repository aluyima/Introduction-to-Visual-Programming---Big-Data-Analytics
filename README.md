CBC Survey Dashboard — Big Data Analytics Assignment
Course
PIS 9204 Big Data Analytics, PhD in Information Systems, Makerere University
Study Context
This repository contains data and visualizations from a field study on the CBC Lesson Plan Generator, a system that supports Ugandan secondary school teachers in CBC-aligned lesson planning. Two surveys were administered across multiple secondary schools in Uganda in early 2025.
Data
survey1_cleaned.csv — Post-introduction survey, N = 103 secondary school teachers. Measures five Technology Acceptance Model constructs: Perceived Usefulness, Perceived Ease of Use, Behavioral Intention, System Quality, and Job Relevance.
survey2_cleaned.csv — Post-implementation survey, N = 69 teachers who generated and taught at least one lesson plan using the system. Measures Pedagogical Content Knowledge Gains, System Quality, Teaching Self-Efficacy, and Life Satisfaction.
Files
cbc_survey_dashboard.html — Self-contained D3 v7 dashboard. No server required. Open in any modern browser.
survey1_cleaned.csv — Survey 1 source data.
survey2_cleaned.csv — Survey 2 source data.
Visualizations
The dashboard contains six D3 charts:

Radar chart — TAM construct means, Survey 1
Donut chart — Subject distribution, Survey 1
Stacked bar chart — Perceived Usefulness Likert distribution, Survey 1
Bubble pack chart — PCK item means, Survey 2
Grouped bar chart — Survey 2 construct comparison
Histogram — Planned lesson delivery dates, Survey 1

How to View
Download cbc_survey_dashboard.html and open the file in a browser. All data is embedded. No internet connection or installation needed.
